¿Qué es Bootstrap?

Bootstrap es un framework de desarrollo front-end gratuito y de código abierto para la creación de sitios web y aplicaciones web. Contiene plantillas basadas en HTML y CSS para tipografía, formularios, botones, tablas, navegación y otros componentes de interfaz; así como extensiones de JavaScript opcionales.

Diseño Responsivo

Bootstrap facilita la creación de sitios web que se adaptan a diferentes tamaños de pantalla.
